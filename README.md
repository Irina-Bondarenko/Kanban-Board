<h1 align="center">GitHub Kanban Board</a> 
<h2>Description</h2> 
<p>A simple GitHub issues viewer presented as a Kanban board</p>

<h2>Installation</h2> 

<ol>
  <li>Install dependencies:</li>
</ol>
<pre><code>npm install
</code></pre>
<ol start="2">
  <li>Start the application:</li>
</ol>
<pre><code>npm run dev
</code></pre>

<p><em>Note: Make sure you are in the <code>kanban_board</code> directory or use <code>cd kanban_board</code> before running the commands.</em></p>

<h2>Testing</h2>
<p>To run tests, use:</p>
<pre><code>npm test
</code></pre>
